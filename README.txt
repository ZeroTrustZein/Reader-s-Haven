Project Overview
A professional Django-based REST API developed for a local bookstore to manage inventory efficiently. This project demonstrates the transition from static data to a dynamic, scalable architecture suitable for real-world business applications.

Key Features
Full CRUD Functionality: Mrs. Smith can Create, Read, Update, and Delete inventory items.

Implemented a Many-to-Many relationship between Books and Authors, allowing for complex literary collaborations.

Search & Filter: Dynamic searching by book title or author name using case-insensitive partial matching (icontains).

Ordering Logic: Inventory can be sorted by price (ascending/descending) and alphabetically by title.

Technical:

Backend: Django

API Framework: Django REST Framework

Database: SQLite
